1. CODE CHANGES FROM PREVIOUS ITERATION
--------------------------------------------------------------------------------
Adds BLOCK_KC (k-dimension blocking, default 128) to the existing BLOCK_NC blocking. Now both the k-loop (reduction dimension) and n-loop are blocked:

```c
for( int p1 = 0; p1 < k0; p1 += BLOCK_KC ) {
  int pb = (p1 + BLOCK_KC <= k0) ? BLOCK_KC : k0 - p1;
```

This ensures that panels of A (m×BLOCK_KC) and B (BLOCK_KC×n) fit in L2/L3 cache during the inner product accumulation, significantly improving data reuse.
